# IoT Based Rain Detection System using ESP32

## Project Overview
This project detects rain using ESP32. When rain is detected, LED and buzzer are activated. The rain status can be monitored through a cloud dashboard.

## Components Used
- ESP32
- Rain Sensor / Push Button
- LED
- Buzzer
- Resistor

## Features
- Rain Detection
- LED Alert
- Buzzer Alert
- Cloud Monitoring
